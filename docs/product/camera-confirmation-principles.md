# Camera Confirmation Principles

Computer vision helps users log faster, but it is never treated as final nutrition truth.

The camera workflow must:

1. Identify likely food components.
2. Return candidates when uncertain.
3. Attach confidence to every detected item.
4. Ask the user to confirm or replace each food.
5. Ask for confirmed weight, serving, household measure, or volume.
6. Ask preparation questions such as fried, baked, grilled, skin on/off, bone-in, added oil, sauce, dressing, cheese, sugar, and condiments.
7. Retrieve nutrients from an authoritative source.
8. Show source and confidence for every result.
9. Avoid saving until the user confirms the analysis.

Recommended language includes "estimated", "needs confirmation", "based on the portion entered", and "nutrition source".
